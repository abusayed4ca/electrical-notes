Megger Testing

Megger testing measures insulation resistance in electrical systems.

Procedure

  1. Disconnect equipment from power.
  2. Connect megger leads to conductor and ground.
  3. Select test voltage.
  4. Run the test for 1 minute.
  5. Record resistance value.

Acceptable Results

  • Higher resistance = better insulation
  • Values usually in Megaohms (MΩ)
